# Rohith Anand — Portfolio

> **Live site →** [rohith-max003.github.io](https://rohith-max003.github.io)

A premium, hand-crafted portfolio website for **Sairama Rohith Anand** — Senior Software Engineer with 5+ years of experience in Java, Spring Boot, AWS, Azure, and AI-augmented engineering.

---

## ✨ Features

- **Particle canvas hero** — animated floating particles with connecting network lines
- **Typewriter effect** — cycling through roles: Senior Software Engineer, Cloud-Native Architect, AI-Augmented Builder, and more
- **Custom cursor** — smooth dot + ring cursor with hover grow animations
- **Scroll progress bar** — gradient progress indicator at the top of the page
- **Animated counters** — numbers count up when scrolled into view
- **Experience timeline** — vertical timeline with hover animations for career history
- **Glassmorphism cards** — backdrop-blur skill and project cards with gradient borders
- **Tech marquee ticker** — scrolling strip of technology badges
- **Mobile hamburger menu** — fully responsive with smooth transitions
- **Scroll reveal animations** — staggered reveal for all major sections
- **Rotating avatar ring** — conic-gradient spinning border with counter-rotation

## 🚀 Tech Stack

Built with **zero frameworks** — pure HTML, CSS, and vanilla JavaScript.

- No React, no Vue, no dependencies
- Google Fonts (Syne, Inter, Fira Code)
- `IntersectionObserver` API for scroll reveal and counter animations
- `requestAnimationFrame` for custom cursor and particle canvas
- GitHub Pages for hosting (auto-deploys on push to main)
